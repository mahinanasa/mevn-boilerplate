Single page app, authentication with jwt, 'active validation' with vee-validate, easy to expand

## Usage
### Installation
```sh
git clone https://github.com/ste7/mevn.git
```
### Serve
To serve the backend
```sh
cd boilerplate/app
npm start
``` 
To serve the frontend
```sh
cd ../public
npm run dev
```
